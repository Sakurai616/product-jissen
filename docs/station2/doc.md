# 要件定義

## メンズ美容支援アプリ「Ikemenize - 自分史上最高の外見へ。」
### 課題  
外見磨きに関心がある男性の増加やルッキズム（外見至上主義）の台頭がある。  
その中で以下のような悩みや課題があると想定。  

- 自分の外見に自信が持てないので、かっこよくしたいが、客観的な改善ポイントやその改善方法が分からない。

- YouTubeやSNSの情報が多すぎて、何をすればよいか分からない・色々試したがいまいち成果が出ない。

- 外見磨きは継続が大切だが、習慣化や記録の仕組みがないのでモチベが上がらず、継続できない。

#### 参考データ・仮説
1.外見磨きに関心がある男性が増加している。
- https://prtimes.jp/main/html/rd/p/000000113.000004342.html

  
2.客観的な改善ポイントや何をすればよいか分からない。  
- [【男性の自分磨き】7割以上が必要性を感じる一方で、実施しているのは5割以下！「ヒゲの濃さ」が外見で気になるポイントとして最多に](https://www.fnn.jp/articles/-/779823)
 
- 数多くの外見磨きに関するYouTubeチャンネルが存在している。  

3.改善方法が分かっても、継続が難しい
- 習慣化には動機付け・視覚化・ゲーミフィケーションなどの支援が必要

4.外見に対する悩みは、自己肯定感やコミュニケーションへの影響も及ぼす 　
- 恋愛・婚活や職場など、社会的な場面での第一印象・自信に影響を与える 　
- 自分に自信が持てないことが、行動の消極性や不安感にもつながる

### ターゲット
- 20〜30代の男性、社会人や大学生で本気でイケメンになりたい人

- 外見改善を通じて自信をつけたい男性

- 恋活や婚活をしていて彼女や結婚相手を見つけたい男性

### 解決策
- ユーザーが自分の顔写真や体型データ、目標（清潔感を出したい・モテたい）を登録

- AIやルールベースで顔や体型の改善ポイントを提示（例：眉・髪型・肌・目・体重）

- 改善に向けたTODOリストと、おすすめ美容サロンやクリニックを提案

- モチベーション維持のために変化記録・習慣トラッカーやRPG要素も導入

### 提供価値
- 「何をすればいいか」が明確になり、迷わず自分に合った外見改善に取り組める

- 外見改善による自己肯定感の向上・人間関係や仕事への好影響

- 自分に合った美容サービスへアクセスでき、時間やお金の無駄を削減

- 外見磨きを楽しく習慣化し、継続できるようにする（ゲーム性・達成感・記録）

### 実装機能
- ユーザー機能
   - ユーザー登録・ログイン
   - プロフィール登録：目標（モテたい・清潔感UPなど）や現状の顔や体型を設定
- 顔・体型分析機能（※顔分析は外部API連携）
   - 顔写真アップロード
   - 顔分析（AI）：髪型・眉・肌・目・髭・パーツ配置等の客観的な評価
   - 体型分析：身長・体重・体脂肪などから現在の状態を把握
   - 改善ポイント抽出：顔・体それぞれの改善可能な領域を提示
- TODOリスト・ゲーム機能：TODOリストを達成するごとに敵に攻撃できる形式のゲーム
   - パーソナライズTODO：分析結果に応じたTODO自動生成
   - 敵キャラとHP表示
   - 美容RPGシステム：進捗によって新しい敵が登場、やりこみ要素を演出
- 美容サロン・クリニック提案機能
   - 美容サロン/クリニック紹介：顔・体型の悩みに応じておすすめを表示（地図連携）
   - お気に入り保存：気になるものをブックマークし、後で確認可能
- 履歴・称号機能
   - TODO履歴・グラフ：達成数、連続記録日数などを表示
   - 敵撃破ログ：倒した敵の一覧や称号を収集できるようにする
   - 顔・体のBefore/After記録：画像履歴で成長を確認（スライダー比較など）
   - 称号・レベルシステム：	成長や習慣を表す称号（例：「保湿マスター」「眉毛戦士」など）を表示
